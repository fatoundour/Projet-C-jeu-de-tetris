# Projet-C-jeu-de-tetris
Ce projet portait sur le developpement d'un jeu de casse-tete Tétris en utilisant les différentes notions 
acquises en c++. C'est un jeu qui consiste à faire déplacer une piéce selon sa position et le type de 
déplacement possible. Il inclut l'utilisation de pattern comme Decor et Visitor.
